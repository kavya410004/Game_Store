'Boo!' is a spooky puzzle game for you!

To solve the puzzles you have to dip the pumpkin in the different paints and wear attributes to mask certain parts of the pumpkin. The puzzles start easy but it gets more difficult! Can you make the correct pumpkin in 42 Halloween themed levels? Complete all the levels to unlock the brand new daily puzzle mode...

Minimun requirements

Operating System: The game requires a mobile device running on Android 4.1 or later, or iOS 8.0 or later.

Storage Space: You need to have enough storage space on your mobile device to install the game. The game has a size of around 50MB, so make sure your device has at least that much free space.

Internet Connection: While "Boo!" doesn't require an internet connection to play, you may need to download the game from the app store, so make sure you have a stable internet connection for the initial download.

Hardware: The game is designed to be played on mobile devices, so you need to have a device with a touch screen and a decent processor to ensure smooth gameplay.
